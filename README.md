## About

This repository is the public GitHub Pages deployment target for ShakalFX.

- Live site: https://mantukin.github.io/shakalfx/
- Source repository: private
- Stack: React, Vite, gif.js, Canvas, Web Workers

## Features

- Upload static images and animated GIFs
- Render ASCII and dither layers in real time
- Stack image, processing, and ASCII passes with blend modes
- Apply post-processing effects for lo-fi and broken-video looks
- Export TXT, PNG, and GIF outputs directly in the browser
